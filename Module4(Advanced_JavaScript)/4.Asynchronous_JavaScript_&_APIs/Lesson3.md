1) What are the 3 things your computer (clients)  might request from a server?
* JSON array of data
* Video stream
* HTML files

2) What is the main job of a server?
Receive a request from the client and return a response.

3) Self Study: what would the 3-digit-server response code be if the server experienced an internal error?
500 Internal Server Error

4) What are other 3 digit server response codes you can think of and what do they mean?
* 200 OK - The request has succeeded.
* 404 Not Found - The server can not find the requested resource.
* 301 Moved Permanently - The requested resource has been assigned a new permanent URI.
* 403 Forbidden - The server understood the request but refuses to authorize it.
* 400 Bad Request - The server could not understand the request due to invalid syntax.

5) What is AJAX?
Asynchronous JavaScript and XML - a technique for creating fast and dynamic web pages by allowing web pages to be updated asynchronously by exchanging small amounts of data with the server behind the scenes.
